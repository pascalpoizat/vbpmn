Installation

In addition to the VBPMN distribution (see here) you need to install:

  • Java RE
    This is needed to transform BPMN files into the PIF format.
    It is not required if you plan to use the PIF format directly.
    Download Java RE
    Note: VBPMN requires JRE\JDK 1.8 and above.
  • Apache Tomcat Server 8.0
    This is needed for deploying and hosting the VBPMN Web Archive (WAR)
     More information about tomcat deployment can be found here
    Download Tomcat 8.0

  • Python distribution and the PyXB library
    This is needed to read PIF files and generate formal description files.
    Download Python 2.7.x Download PyXB 1.1.1
    Warning: Python 2.7.x and PyXB 1.1.1 should be used.
    Newer versions may not work correctly with VBPMN.
  • CADP toolbox
    This is needed to perform formal analyses on the formal description files.
    Download CADP
